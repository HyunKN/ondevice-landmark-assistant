# Contribution boundary

This portfolio separates product/system design from source-code implementation.

| Area | My role | What is shown here |
|---|---|---|
| Project direction and scope | Co-defined with the team | Shared project context; not claimed as an individual decision |
| Model experiments | Designed and implemented | Training, evaluation, loss, configuration matrix, export and validation source |
| Sprint 1 prototype | Designed and implemented | The tracked Streamlit prototype snapshot |
| Model-to-app integration flow | Designed from the Sprint 1 prototype | The handoff flow and contract documentation |
| Final Flutter/Android app implementation | Implemented by the team; not claimed as my individual work | Exact team commit/file references, not copied team source |
| App UI design and implementation | Handled by the team; not my contribution | Shown only as project-result screenshots |
| Model artifact integration | Implemented and debugged integration changes | Contract scripts, semantic artifact generator and the Android cache-fix patch |
| Auth, notifications and suggestions | Not my contribution | Excluded |

Git history is used here as evidence of code changes, not as proof of who originated a product or architecture decision. The Sprint 1 prototype and handoff documents provide the design lineage; the final app repository provides the implementation history.
